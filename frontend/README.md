# Employee Shift Board - Frontend

A professional React frontend for the Employee Shift Board application.

## Features

- Login authentication with JWT
- Dashboard with shift management
- Admin-only shift assignment form
- Shift table with filtering capabilities
- Responsive UI with TailwindCSS

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will run on `http://localhost:3000`

## Build

```bash
npm run build
```

## Default Credentials

- **Admin**: username: `admin`, password: `adminpass`
- **User**: username: `user`, password: `userpass`
